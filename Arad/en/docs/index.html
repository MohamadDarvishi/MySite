Questions and answers about Arad font

What is Arad font license?

Arad font, like other open source fonts, is licensed by OFL. Anyone can use it for any purpose.

What languages ​​does Arad font support?

Arad font supports Farsi, Arabic, Türkcə, Urdu, Kurdi, Mazeruni and English languages.

What is the difference between Arad font and Shabnam font?

Although the Shabnam font was still in use, it was an archived project and its bugs were not fixed and it was not as developed as Vazirmatn Font. Arad font is made based on the structure and purpose of Shabnam font, so that it is bigger and does not have the defects of Shabnam. Arad font has been redesigned and has a better design, more weight, new modes, support for more languages ​​and...

In what category is the Arad font?

Arad font is of Sans type, but it is placed in the category of geometric fonts in Persian fonts.

How many weight/dots style does Arad font have?

10 weights and three dot styles. A total of 30 modes

Weight:
  • Thin
  • ExtraLight
  • Light
